# Memory Manager Simulator

## Descrizione

Questo progetto consiste nella realizzazione di una **applicazione web** sviluppata con **HTML, CSS, JavaScript** che permette di simulare la gestione della memoria centrale.

Questa applicazione consente di:
- Visualizzare la RAM (come rettangolo verticale)
- Simulare la seguente tecnica di allocazione: **Dynamic Partitions**
- Inserire i processi
- Calcolare lo spazio libero / frammentazione
- Speigazione della tecnica scelta
